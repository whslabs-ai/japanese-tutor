Chapter 1: Basic Forms
- です: Polite copula. X です = This is X
- は: Topic marker. X は Y です = As for X, it is Y
- の: Possessive. X の Y = X's Y

Chapter 2: Basic Questions
- これ/この/ここ: Near speaker
- それ/その/そこ: Near listener
- あれ/あの/あそこ: Far from both
- Question words: だれ、どこ、なに + が
- も: Also (replaces は)
- Negatives: じゃないです
- ね/よ: Confirmation/emphasis

Example Grammar Analysis:
Given sentence:
私は昨日図書館で勉強していました。
(I was studying at the library yesterday)

1. Pattern Recognition:
   - ている (Chapter 7: States)
     • Basic: 勉強する → 勉強している
     • Past form: 勉強していました
   - で particle (Chapter 3: Basic Particles)
     • Marks location of action

2. Pattern Usage:
   - ている: Shows ongoing state/action
     • Present: 勉強しています
     • Past: 勉強していました
     • Casual: 勉強してる
   - で: Indicates where action occurs
     • 図書館で = at the library

3. Pattern Analysis:
   - ている adds continuous aspect
   - Past tense ていました shows action was ongoing
   - Polite form appropriate for most situations
   - Common mistake: *勉強をしていました (を not needed)

Similar examples:
- 私は今本を読んでいます。(reading now)
- 友達は家で寝ていました。(was sleeping)
- 先生は教室で待っています。(is waiting)
